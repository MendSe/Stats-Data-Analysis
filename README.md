A collection of solutions and notebooks for the course "Statistics and Data Analysis".

The work in this repository focuses on applying statistical and analytical methods to real datasets, combining theoretical concepts with practical coding and visualization.

## Topics Covered

- **Probability theory**  
  Fundamental probability concepts, random variables, expectation, variance, and probabilistic reasoning.

- **Statistical distributions**  
  Study and application of common distributions such as binomial, negative binomial, normal, and multinomial distributions.

- **Descriptive statistics**  
  Summarizing datasets using measures such as mean, variance, standard deviation, quantiles, and other basic statistical indicators.

- **Exploratory data analysis (EDA)**  
  Investigating datasets through summaries and visualizations in order to understand structure, detect patterns, identify anomalies, and generate insights.

- **Data visualization**  
  Creating plots and graphical representations of data, including histograms, scatter plots, boxplots, correlation heatmaps, and other visual tools for analysis.

- **Correlation and dependence analysis**  
  Measuring relationships between variables using methods such as Pearson correlation, Spearman correlation, and Kendall rank correlation.

- **Statistical inference**  
  Drawing conclusions from data through estimation, interpretation, and reasoning under uncertainty.

- **Hypothesis testing**  
  Applying statistical tests to evaluate claims about data and determine whether observed patterns are statistically significant.

- **Mixture models and probabilistic modeling**  
  Working with probabilistic models such as Gaussian mixture models and using algorithmic methods for estimation and analysis.

- **Real-world data analysis**  
  Applying statistical and computational tools to real datasets in order to extract insights, test hypotheses, and better understand practical data-driven problems.
